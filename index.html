Settings

This app runs entirely in your browser. To connect Spotify you need a (free) Spotify Developer application — create one at developer.spotify.com/dashboard, add this page's URL as a Redirect URI, and paste the Client ID below.

If a track has no lyrics on Spotify, the app checks Musixmatch, then LRCLIB (a free synced-lyrics database, no key needed).
